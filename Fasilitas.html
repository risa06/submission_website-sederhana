Deskripsi Nama lengkap saya adalah Risa Mahardika Sari, biasanya dipanggil Risa. Saya adalah mahasiswa D4 TRPL (Teknologi Rekayasa Perangkat Lunak) SV UGM angkatan 2019. Saya tertarik dengan perangkat lunak karena waktu saya berumur kira-kira 14 tahun saya sering memainkan game di gawai dan laptop/PC. Saya sering bertanya-tanya bagaimana cara membuat game tersebut, bagaimana membuat tampilan game agar terlihat menarik, dan bagaimana teknologi itu dikembangkan menjadi sebuah aplikasi. Atas saran dan pilihan orang tua saya, akhirnya saya memilih berkuliah di D4 TRPL SV UGM.
Saya tertarik dengan pemrograman web dan saya ingin mempelajari lebih dalam lagi tentang pemrograman web. Oleh karena itu, saya mengikuti program SIB (Studi Independen Bersertifikat) Kampus Merdeka X Dicoding Indonesia Batch Kedua, dan mendaftar pada paket belajar Pengembang Front-End Web dan Back-End.
